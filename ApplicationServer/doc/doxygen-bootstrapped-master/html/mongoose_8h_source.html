1979  *

1981  * is passed as
1982  * `struct http_message` through the handler's `void *ev_data` pointer.
1983  * - MG_EV_HTTP_MULTIPART_REQUEST: A multipart POST request has received.
1984  * This event is sent before body is parsed. After this user
1985  * should expect a sequence of MG_EV_HTTP_PART_BEGIN/DATA/END requests.
1986  * This is also the last time when headers and other request fields are
1987  * accessible.

1990  * - MG_EV_HTTP_CHUNK: HTTP chunked-encoding chunk has arrived.
